# NIST 800-53 Compliance Mapping (KFG-v3)

| Control ID | Control Name | Technical Implementation in KFG-v3 |
| :--- | :--- | :--- |
| **AC-3** | Access Enforcement | **GCP Workload Identity** ensures only authorized K8s pods can access Vertex AI. |
| **AU-2** | Event Logging | **OpenTelemetry** traces provide a full audit trail of every AI-driven decision. |
| **SC-7** | Boundary Protection | **Private VPC + Cloud NAT** isolates the Nervous System from the public internet. |
| **SI-4** | System Monitoring | **Prometheus + Grafana** metrics track anomalous signal spikes in the stream. |
| **IA-2** | Identification & Auth | **IAM Roles** (via Terraform) enforce the Principle of Least Privilege. |
